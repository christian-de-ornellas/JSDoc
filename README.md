# JsDoc
This app have all docs JS.

## Installation

### Clone repository
~~~
git clone https://github.com/christian-de-ornellas/JSDoc.git
~~~

### Install dependencies

~~~
yarn
~~~

### Init project
~~~
react-native start
~~~
### Emulator an Android
~~~
react-native run-android
~~~
### Emulator an IOS
~~~
react-native run-ios
~~~
